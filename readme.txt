Education & Career Success – Streamlit Web Application

This web explores the relationship between age, gender, entrepreneurship and career outcomes through interactive elements.

Key Features:
- A user-friendly homepage introducing the topic.
- Multiple interactive charts, including:
  • Gender Distribution
  • Field of Study
  • Job Offers and Entrepreneurship 
- Filter options to interact with and explore the data.
- Custom design elementsusing Streamlit's layout system and CSS styling.
